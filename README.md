# .github
Скачать видео из пинтерест — Uniloader помогает сохранять ролики Pinterest в MP4 и оригинальном качестве. Просто отправьте ссылку боту в Telegram, выберите формат и получите файл на телефон или компьютер. Не нужно устанавливать дополнительные приложения. Работает на Android, iOS и ПК для личного просмотра.
